# Centrium - Быстрый старт

## Запуск проекта

### 1. Установка зависимостей
```powershell
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

### 2. Применение миграций
```powershell
.venv\Scripts\python.exe manage.py migrate
```

### 3. Создание тестовых пользователей
```powershell
.venv\Scripts\python.exe manage.py create_init_data
```

Создаст:
- `admin / admin` (администратор)
- `operator / operator` (оператор)
- `client / client` (клиент)

### 4. Создание демо-тикетов
```powershell
.venv\Scripts\python.exe manage.py create_demo_tickets
```

Создаст 5 тестовых тикетов с сообщениями для демонстрации.

### 5. Запуск сервера
```powershell
.venv\Scripts\python.exe manage.py runserver
```

Откройте: http://127.0.0.1:8000/

## Структура проекта

### Клиентская часть
- `/` - Главная страница с hero, преимуществами, спидтестом
- `/chat/` - Чат с AI-ассистентом
- `/login/` - Вход в систему
- `/register/` - Регистрация

### Панель оператора
- `/tickets/operator/` - Список всех тикетов
- `/tickets/operator/<id>/` - Детальная страница тикета

### Дашборд (только для админа)
- `/tickets/dashboard/` - Метрики и статистика

## Тестовые аккаунты

| Логин | Пароль | Роль | Доступ |
|-------|--------|------|--------|
| admin | admin | Администратор | Всё + дашборд |
| operator | operator | Оператор | Панель оператора |
| client | client | Клиент | Чат |

## Возможности

### AI-ассистент
- Автоматическая классификация обращений
- Генерация ответов на типовые вопросы
- Анализ изображений (если приложены)
- Поддержка русского и казахского языков

### Спидтест
- Проверка скорости интернета
- Автоматическое предложение открыть чат при низкой скорости
- Демо-режим (работает без реального API)

### Панель оператора
- Просмотр всех тикетов
- Фильтрация по статусу, категории, приоритету
- Генерация AI-ответов для операторов
- История переписки с клиентами

### Дашборд
- Общее количество тикетов
- % авто-решённых обращений
- Среднее время до первого ответа
- Распределение по категориям (график)

## Дизайн

Проект выполнен в стиле Stripe:
- Чистая типографика (Inter)
- Минималистичный дизайн
- Тёмный header
- Плоские цвета (без градиентов)
- Хорошие отступы и иерархия

Цветовая палитра Казахтелеком:
- Основной: #00355E (тёмно-синий)
- Акцент: #00AEEF (голубой)
- Фон: #ffffff (белый)
- Поверхность: #f6f9fc (светло-серый)

## Troubleshooting

### Ошибка "no such table"
Запустите миграции:
```powershell
.venv\Scripts\python.exe manage.py migrate
```

### Нет тикетов в панели оператора
Создайте демо-данные:
```powershell
.venv\Scripts\python.exe manage.py create_demo_tickets
```

### AI не отвечает
Проверьте `.env` файл и наличие `GEMINI_API_KEY`. Если ключа нет, система работает в демо-режиме с заготовленными ответами.

### Спидтест не работает
Это нормально для демо. Спидтест использует внешние ресурсы и может не работать в некоторых сетях. В таком случае показывается демо-значение.
